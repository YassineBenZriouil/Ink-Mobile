# 🖋️ Ink

A minimal notes & to-do list app — simple by design.

## About

Ink is a clean, distraction-free app for capturing notes and managing tasks. Built with simplicity at its core, it focuses on what matters: your thoughts.

## Platforms

- **Mobile** — React Native (Android & iOS)
- **Web** — [ink-web](https://ink-web.app) *(coming soon)*

## Getting Started

```bash
# Install dependencies
npm install

# Run on Android
npm run android

# Run on iOS
npm run ios
```

## Tech Stack

- React Native
- TypeScript
- React Navigation
